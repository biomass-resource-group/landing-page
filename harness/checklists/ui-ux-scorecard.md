# UI/UX Scorecard (1-5)

Scoring scale:
- **1** = unacceptable / broken
- **2** = major issues
- **3** = acceptable with notable gaps
- **4** = strong
- **5** = exemplary

A merge-ready result requires:
- Every category average >= 4.0
- No individual criterion <= 2

## 1) Clarity & Messaging

- [ ] Hero communicates what BRG offers within 5 seconds.
- [ ] Supporting copy is concise, specific, and non-redundant.
- [ ] Terminology is consistent across pages.
- [ ] User can identify the next best action immediately.

Category score: __/5

## 2) Visual Hierarchy & Brand Consistency

- [ ] Heading levels create clear reading flow.
- [ ] Spacing rhythm is consistent section-to-section.
- [ ] Typography usage matches brand patterns.
- [ ] CTA visual prominence is consistent and intentional.

## 3) Interaction & Navigation UX

- [ ] Primary navigation is predictable and easy to scan.
- [ ] CTA paths are clear from each key route.
- [ ] No interaction ambiguity (button vs link semantics, affordance clarity).
- [ ] Mobile menu and section jumps are frictionless.

## 4) Accessibility & Inclusive Design

- [ ] Keyboard navigation reaches all interactive elements.
- [ ] Focus styles are visible and consistent.
- [ ] Color contrast is sufficient for text and controls.
